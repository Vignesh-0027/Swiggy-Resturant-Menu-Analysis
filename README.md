# Swiggy-Resturant-Menu-Analysis
Project Overview
This project performs an Exploratory Data Analysis (EDA) on a dataset containing menu items, pricing, and customer ratings for various restaurants in Bengaluru, Karnataka, sourced from Swiggy. The goal is to understand market trends, pricing strategies across different food categories, and identifying top-performing restaurants in India's "Silicon Valley."

Dataset Description
The dataset consists of 3,101 rows and 9 columns, providing a granular look at the food delivery ecosystem in specific Bengaluru localities.

Key Features:

State/City: Location metadata (focused on Bengaluru).

Restaurant Name: The name of the eatery.

Location: Specific neighborhood (e.g., Rajarajeshwari Nagar, Kengeri).

Category: Food classification (e.g., Main Course, Desserts, Beverages).

Dish Name: The specific item sold.

Price (INR): Cost of the item.

Rating: Customer rating (0.0 to 5.0).

Rating Count: Number of reviews received for that specific dish.

Technical Stack
Language: Python

Libraries: Pandas (Data Manipulation), Matplotlib & Seaborn (Data Visualization), NumPy (Numerical Analysis).
